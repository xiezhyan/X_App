# X_App

#### 使用
> 仓库地址
```
allprojects {
    repositories {
        google()
        jcenter()
        mavenCentral()
        maven { url "https://jitpack.io" }
    }
}
```
> 引用

[![](https://jitpack.io/v/xiezhyan/X_App.svg)](https://jitpack.io/#xiezhyan/X_App)
```
implementation 'com.github.xiezhyan:X_App:Tag'
```

#### 介绍
整理学习到的Android技能成通用框架
